    Builds a tree from an iterator of pairs (numberOfChildren, value), where:

    Builds a tree from an iterator of pairs (numberOfChildren, value), where:

    • value is the value of a new node, and
    • numberOfChildren is a number of preceding elements in the list to become direct subtrees of the node.
    Note

    - Values of subtrees must always precede the value of a parent node, and appear in the reverse order.

    • The sum of all numberOfChildren values must be the size of the list minus one.
